## Getting Started

Open `index.html` in your browser to explore the festival website.

## Features

- Event browsing with search & quick view
- Food stall directory with category filtering
- Schedule with day tabs
- Gallery with lightbox and category filters
- Ticket booking with tier selection
- Contact form with FAQ accordion
- Login & password reset pages
- Responsive design
